NTeam

NTeam is a Joomla! module that displays the list of team members in a fancy way.

Features:

  • Can add up to 100 members.
  • Adjustable height and width of images.
  • Can change the width and color of border.
  • Automatically optimizes images according to the width and height specified.
  • Adjustable no. of members displayed per row.
  • Adjustable font size.
  • Fully CSS customizable.
  • Link the members to their social profiles.
  • Option to open link in new tab or same window.
